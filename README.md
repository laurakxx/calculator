# Calculator

Консольный калькулятор на Java с поддержкой:
- операций `+`, `-`, `*`, `/`
- чисел от 0 до 1000
- ввода чисел цифрами и словами
- команд `history`, `last`, `clear`, `help`, `exit`
- истории последних 10 операций


## Примеры ввода

- `2 + 3`
- `10 на 4` - 10*4
- `сто пять плюс двести`
- `пять делить на два` - вывод double
- `11-50` - вывод отрицательного числа
- `history`
- `last`
- `clear`
- `exit`
